                    <div class="mb-6 border rounded-lg border-gray-200 bg-gray-50">
                        <div class="collapsible-header flex justify-between items-center p-4 cursor-pointer" id="collapsible-header">
                            <h3 class="font-semibold text-lg text-sky-600">Bedrock Acceleration Calculation Logic</h3>
                            <i class="fas fa-chevron-down collapsible-icon text-gray-500" id="collapsible-icon"></i>
                        </div>
                        <div class="collapsible-content" id="collapsible-content">
                            <div class="p-4 border-t prose max-w-none border-gray-200">
                                <p>The calculation method depends on the selected <strong>Analysis Mode</strong>. The logic has been updated to allow bedrock acceleration to exceed seed acceleration, modeling potential soil amplification effects.</p>
                                <h4 class="text-lg font-semibold mt-4 mb-2 text-amber-600">Mode 1: Segment & Average</h4>
                                <p>This mode requires both <strong>Seed</strong> and <strong>Numerical</strong> acceleration data. The <code>Effective Factor</code> for each segment represents the soil's transfer function (amplification or attenuation). The final bedrock acceleration is calculated by applying the inverse of this effect to the numerical (surface) acceleration.</p>
                                
                                <p><strong>Bedrock Acceleration Logic (per step):</strong><br>
                                <code>Bedrock Accel. = Numerical Accel. / Effective Factor</code></p>
                                <p>This single formula correctly handles both scenarios:</p>
                                <ul class="list-disc pl-5 mt-2 space-y-2">
                                    <li>If <strong>Effective Factor > 1</strong> (amplification), the bedrock acceleration will be scaled down from the numerical acceleration.</li>
                                    <li>If <strong>Effective Factor < 1</strong> (attenuation), the bedrock acceleration will be scaled up from the numerical acceleration.</li>
                                    <li>If <strong>Effective Factor is Negative</strong> or near-zero, a fallback value is used to ensure stability.</li>
                                </ul>
                                
                                <h4 class="text-lg font-semibold mt-6 mb-2 text-indigo-600">Mode 2: Direct Scaling</h4>
                                <p>This mode provides two layers of control: broad adjustments across four main periods, plus an additional set of user-defined specific time ranges for fine-tuning. For any time step, the most specific adjustment (a custom range) will take precedence over the broader period adjustment. Numerical data is ignored.</p>
                                 <ol class="list-decimal pl-5 space-y-3">
                                     <li><strong>Factor Precedence:</strong> A factor map is created for every time step. First, factors from the four main periods are applied. Then, factors from any defined specific adjustment ranges are applied, overwriting the main period factors for those specific times.</li>
                                    <li><strong>Bedrock Acceleration Logic (per step):</strong>
                                        <ul class="list-disc pl-5 mt-2 space-y-2">
                                            <li><code>Bedrock Accel. = Seed Accel. * Final Factor from Map</code></li>
                                        </ul>
                                    </li>
                                </ol>
                                <h4 class="text-lg font-semibold mt-6 mb-2 text-rose-600">Restraint Factors & Limitations (Applicable to Both Modes):</h4>
                                <ul class="list-disc pl-5 space-y-3">
                                    <li><strong>Amplification Enabled:</strong> The previous cap preventing bedrock acceleration from exceeding seed acceleration has been removed. This allows the final result to reflect higher intensity, which can occur due to soil amplification effects during a seismic event.</li>
                                    <li><strong>Segment & Average Mode Cap:</strong> In Segment & Average mode, if the effective factor exceeds 1, it is capped at a maximum value of 2 to prevent excessive attenuation.</li>
                                    <li><strong>Data Consistency:</strong> For Segment & Average mode, both seed and numerical acceleration data must have the same number of time steps. Mismatched data sets will prevent bedrock motion generation.</li>
                                    <li><strong>Numerical Stability:</strong> Division by zero is prevented by ignoring time steps where the seed acceleration value is below a threshold (1e-9).</li>
                                </ul>
                            </div>

        function generateWaveform(samples, duration) {
            const dt = duration / samples;
            const t = Array.from({ length: samples }, (_, i) => i * dt);
            return t.map(ti => {
                const noise = (Math.random() - 0.5) * 0.1;
                const p1 = Math.exp(-0.6 * ti) * Math.sin(2 * Math.PI * 2.0 * ti);
                const p2 = 0.6 * Math.exp(-0.8 * (ti - 2)) * Math.sin(2 * Math.PI * 5.0 * (ti - 2)) * (ti > 2 ? 1 : 0);
                return p1 + p2 + noise;
            });
        }

        function applyFilter(signal, ampFactor, damping) {
            const dt = 8 / signal.length;
            const t = Array.from({ length: signal.length }, (_, i) => i * dt);
            return signal.map((val, i) => val * ampFactor * Math.exp(-damping * t[i]));
        }

        function renderPath(id, signal, width, height) {
            const y = (s, h) => {
                const max = Math.max(...s.map(Math.abs)) * 1.2 || 1;
                return s.map(v => h * 0.5 * (1 - v / max));
            };
            const yPoints = y(signal, height);
            const xStep = width / (signal.length - 1);
            let d = `M 0 ${yPoints[0]}`;
            for (let i = 1; i < signal.length; i++) d += ` L ${i * xStep} ${yPoints[i]}`;
            const path = document.getElementById(id);
            if (path) path.setAttribute("d", d);
        }

                });

                    const factor = parseFloat(adj.factor);

                    const startStep = Math.round(start / timeInterval);
                    const endStep = Math.round(end / timeInterval);

                    for (let j = startStep; j < endStep; j++) {
                        if (j >= 0 && j < factorMap.length) {
                            factorMap[j] = factor;
                        }
                    }
                });
                
                return appState.seedData.map((val, i) => val * factorMap[i]);
            };

            const adjustLayerOrder = (datasetIndex, action) => {

                if (currentIndex === -1) return;

                if (action === 'bring-forward') currentOrder.splice(Math.max(0, currentIndex - 1), 0, datasetIndex);

                        if (e.clientY > offset) {
                            target.style.borderBottom = '3px solid var(--accent-primary)';

                        e.stopPropagation();

                        target.style.borderBottom = '';

                        const droppedOriginalIndex = parseInt(e.dataTransfer.getData('text/plain'));
                        const targetOriginalIndex = parseInt(target.dataset.originalIndex);

                        const fromPos = appState.sequentialSegmentOrder.indexOf(droppedOriginalIndex);

                        const bounding = target.getBoundingClientRect();

                        appState.sequentialSegmentOrder.splice(e.clientY > offset ? toPos + 1 : toPos, 0, movedItem);

                    });
                });

            function handleSimilarityFileUpload(files, type) {
                if (type === 'seed' && files.length > 0) {
                    const reader = new FileReader();
                    reader.onload = e => {
                        const parsed = parseSimilarityData(e.target.result, files[0].name);
                        if (parsed.isValid) {
                            appState.similaritySeedData = parsed.data;
                            alert(`Reference data "${files[0].name}" loaded successfully.`);
                        } else {
                            alert(`Error parsing reference file: ${parsed.errorMessage}`);
                        }
                    };
                    reader.readAsText(files[0]);
                } else if (type === 'multi') {
                    Array.from(files).forEach(file => {
                         if (!appState.similarityFiles.some(f => f.name === file.name)) {
                            appState.similarityFiles.push({ file, name: file.name, data: null, isReference: false });
                         }
                    });
                    renderSimilarityFileList();
                }
            }
            
            function parseSimilarityData(content, fileName) {
                 try {
                    const lines = content.split('\n').filter(line => line.trim() !== '');
                    if (lines.length === 0) return { isValid: false, errorMessage: 'Empty file' };
                    let data = [];
                    const isCsv = fileName.toLowerCase().endsWith('.csv');
                    let startIndex = isCsv && (lines[0].toLowerCase().includes('time') || lines[0].toLowerCase().includes('accel')) ? 1 : 0;
                    
                    for (let i = startIndex; i < lines.length; i++) {
                        const parts = lines[i].trim().split(/[\s,]+/);
                        const value = parseFloat(parts[parts.length - 1]);
                        if (!isNaN(value)) data.push(value);
                    }

                    if (data.length === 0) return { isValid: false, errorMessage: 'No valid numeric data found.' };
                    appState.similarityTotalSteps = parseInt(document.getElementById('sim-totalSteps').value, 10);
                    return { isValid: true, data: data.slice(0, appState.similarityTotalSteps) };
                } catch (error) {
                    return { isValid: false, errorMessage: error.message };
                }
            }

            async function processSimilarityFiles() {
                if (!appState.similaritySeedData) { alert('Please upload reference data first.'); return; }
                if (appState.similarityFiles.length === 0) { alert('Please upload model output files.'); return; }
                
                for (const fileObj of appState.similarityFiles) {
                    if (!fileObj.data) {
                        const content = await fileObj.file.text();
                        const parsed = parseSimilarityData(content, fileObj.name);
                        if (parsed.isValid) fileObj.data = parsed.data;
                        else { alert(`Failed to process ${fileObj.name}: ${parsed.errorMessage}`); return; }
                    }
                }
                renderSimilarityFileList();

                appState.similarityCurrentResults = calculateSimilarityMetrics();
                displaySimilarityResults(appState.similarityCurrentResults);
                displayImprovementTable(appState.similarityCurrentResults);
                renderEvaluationRadarChart(appState.similarityCurrentResults);
                renderEvaluationBarChart(appState.similarityCurrentResults);
                setupRecommendationEngine(appState.similarityCurrentResults);
            }

            function calculateSimilarityMetrics() {
                const results = appState.similarityFiles.map(fileObj => {
                    if (!fileObj.data) return null;
                    const len = Math.min(appState.similaritySeedData.length, fileObj.data.length);
                    const ref = appState.similaritySeedData.slice(0, len);
                    const pred = fileObj.data.slice(0, len);
                    return {
                        fileName: fileObj.name, isReference: fileObj.isReference,
                        nrmse: calculateNRMSE(ref, pred), ccc: calculateCCC(ref, pred),
                        crossCorrelation: calculateCrossCorrelation(ref, pred), dtw: calculateDTW(ref, pred)
                    };
                }).filter(Boolean);
                results.sort((a, b) => b.ccc - a.ccc);
                results.forEach((res, i) => res.rank = i + 1);
                return results;
            }

                    row.innerHTML = `<td>${fileNameDisplay}</td><td>${res.nrmse.toFixed(4)}</td><td>${res.ccc.toFixed(4)}</td><td>${res.crossCorrelation.toFixed(4)}</td><td>${res.dtw.toFixed(2)}</td><td>#${res.rank}</td>`;

            function displayImprovementTable(results) {

                if (!baseline) { document.getElementById('sim-improvementSection').classList.add('hidden'); return; }
                
                const tableBody = document.getElementById('sim-improvementTableBody');

                    const nrmseImp = ((baseline.nrmse - res.nrmse) / baseline.nrmse) * 100;
                    const cccImp = ((res.ccc - baseline.ccc) / baseline.ccc) * 100;
                    const crossCorrImp = ((res.crossCorrelation - baseline.crossCorrelation) / baseline.crossCorrelation) * 100;

                        <td class="${nrmseImp > 0 ? 'improvement-positive' : 'improvement-negative'}">${nrmseImp.toFixed(2)}%</td>
                        <td class="${cccImp > 0 ? 'improvement-positive' : 'improvement-negative'}">${cccImp.toFixed(2)}%</td>
                        <td class="${crossCorrImp > 0 ? 'improvement-positive' : 'improvement-negative'}">${crossCorrImp.toFixed(2)}%</td>

            function calculateAndDisplayRecommendation(results) {
                if (!results || results.length === 0) return;
                const weights = {
                    nrmse: parseFloat(document.getElementById('sim-nrmseWeight').value) / 10,
                    ccc: parseFloat(document.getElementById('sim-cccWeight').value) / 10,
                    crossCorrelation: parseFloat(document.getElementById('sim-crossCorrelationWeight').value) / 10,
                    dtw: parseFloat(document.getElementById('sim-dtwWeight').value) / 10,
                };
                const metrics = { nrmse: results.map(r => r.nrmse), ccc: results.map(r => r.ccc), crossCorrelation: results.map(r => r.crossCorrelation), dtw: results.map(r => r.dtw) };
                const minMax = {};
                for (const key in metrics) minMax[key] = { min: Math.min(...metrics[key]), max: Math.max(...metrics[key]) };
                let bestFile = null, maxScore = -Infinity;

                results.forEach(res => {
                    const normNrmse = (minMax.nrmse.max - minMax.nrmse.min) > 1e-9 ? (minMax.nrmse.max - res.nrmse) / (minMax.nrmse.max - minMax.nrmse.min) : 1;
                    const normCcc = (minMax.ccc.max - minMax.ccc.min) > 1e-9 ? (res.ccc - minMax.ccc.min) / (minMax.ccc.max - minMax.ccc.min) : 1;
                    const normCrossCorr = (minMax.crossCorrelation.max - minMax.crossCorrelation.min) > 1e-9 ? (res.crossCorrelation - minMax.crossCorrelation.min) / (minMax.crossCorrelation.max - minMax.crossCorrelation.min) : 1;
                    const normDtw = (minMax.dtw.max - minMax.dtw.min) > 1e-9 ? (minMax.dtw.max - res.dtw) / (minMax.dtw.max - minMax.dtw.min) : 1;
                    const totalScore = (normNrmse * weights.nrmse) + (normCcc * weights.ccc) + (normCrossCorr * weights.crossCorrelation) + (normDtw * weights.dtw);
                    if (totalScore > maxScore) { maxScore = totalScore; bestFile = res; }
                });

                const resultSection = document.getElementById('sim-recommendationResultSection');
                if (bestFile) {
                    document.getElementById('sim-recommendationContent').innerHTML = `<div class="file-name">${bestFile.fileName}</div><p>This model provides the best overall match based on your preferences.</p>`;
                    resultSection.classList.remove('hidden');
                } else {
                    resultSection.classList.add('hidden');
                }
            }

            // Metric Calculation Helpers
            function calculateNRMSE(a, p) { const e = Math.sqrt(a.reduce((s, v, i) => s + (v - p[i]) ** 2, 0) / a.length); const r = Math.max(...a) - Math.min(...a); return r > 1e-9 ? e / r : 0; }
            function calculateCCC(a, p) { const n = a.length, mA = a.reduce((s, v) => s + v, 0) / n, mP = p.reduce((s, v) => s + v, 0) / n; const c = a.reduce((s, v, i) => s + (v - mA) * (p[i] - mP), 0) / n; const vA = a.reduce((s, v) => s + (v - mA) ** 2, 0) / n; const vP = p.reduce((s, v) => s + (v - mP) ** 2, 0) / n; const d = vA + vP + (mA - mP) ** 2; return d > 1e-9 ? (2 * c) / d : 0; }
            function calculateCrossCorrelation(a, p) { const n = a.length, mA = a.reduce((s, v) => s + v, 0) / n, mP = p.reduce((s, v) => s + v, 0) / n; const num = a.reduce((s, v, i) => s + (v - mA) * (p[i] - mP), 0); const dA = Math.sqrt(a.reduce((s, v) => s + (v - mA) ** 2, 0)); const dP = Math.sqrt(p.reduce((s, v) => s + (v - mP) ** 2, 0)); return (dA * dP) > 1e-9 ? num / (dA * dP) : 0; }
            function calculateDTW(ts1, ts2) { const n = ts1.length, m = ts2.length; const dtw = Array(n + 1).fill(null).map(() => Array(m + 1).fill(Infinity)); dtw[0][0] = 0; for (let i = 1; i <= n; i++) { for (let j = 1; j <= m; j++) { const c = Math.abs(ts1[i - 1] - ts2[j - 1]); dtw[i][j] = c + Math.min(dtw[i - 1][j], dtw[i][j - 1], dtw[i - 1][j - 1]); } } return dtw[n][m]; }
            function getRankClass(rank) { return { 1: 'rank-1', 2: 'rank-2', 3: 'rank-3' }[rank] || ''; }
